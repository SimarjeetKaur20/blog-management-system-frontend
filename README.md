This is the frontend of the Blog Management System built using **HTML, CSS, and JavaScript**.  
It connects to a Node.js + MongoDB backend API and allows users to register, log in, and manage their blog posts.

## Features
- User registration & login (JWT-based)
- Create and view blog posts
- Search and pagination
- Protected dashboard using token authentication

## Tech Stack
- HTML
- CSS
- JavaScript (Fetch API)

## How to Run
1. Start the backend server on `http://localhost:3000`
2. Open `index.html` using Live Server
3. Login and access the dashboard
